# Morse Code Translator — Interstellar Edition

A simple Morse Code translator with an Interstellar-inspired design.

This project allows users to translate between:

* English
* Arabic
* Morse Code

Built with a cinematic space-themed interface, smooth animations, and a responsive design for all devices.

## Features

* Real-time translation
* Arabic & English support
* Morse Code conversion
* Responsive UI
* Interstellar-inspired visuals

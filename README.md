# Morse Code Translator — Interstellar Edition

Cinematic Morse code transceiver for **Arabic** and **English** — encode, decode, transmit, and learn signals in a premium space-themed UI.

## Features

- Arabic ↔ Morse and English ↔ Morse (bidirectional)
- Live translation, audio playback, oscilloscope visualization
- 3D cosmic background (React Three Fiber)
- Signal history, favorites, download & share
- Responsive · dark luxury UI

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
npm run preview
```
